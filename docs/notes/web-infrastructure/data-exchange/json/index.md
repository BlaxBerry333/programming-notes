# JSON

![](/static/skill-images/web-infrastructure--protobuf.png)

> JavaScript Object Notation

JSON 是一种轻量级的数据交换格式，文件后缀名是`.json`

是 JavaScript 的一个子集，但其描述的数据结构可在不同语言、数据流中使用

## 基本使用

```json
{
    "键": 值,
    "键": 值
}
```

## 值的数据类型

### 字符串

```json
{
  "键": "字符串"
}
```

---

### 数值

值可为整数或浮点数

```json
{
  "键": 111,
  "键": 11.11
}
```

---

### 布尔值

```json
{
  "键": true,
  "键": false
}
```

---

### null

```json
{
  "键": null
}
```

---

### 数组

```json
{
  "键": [数组元素, 数组元素]
}
```

---

### 对象

```json
{
  "键": {
    "键": 值,
    "键": 值
  }
}
```
