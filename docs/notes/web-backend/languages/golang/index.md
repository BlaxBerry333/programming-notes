# Golang

![](/static/skill-images/web-backend--golang.png)

## 下载安装

::: code-group

```zsh [asdf]
# 1. 下载 asdf 插件与的版本         // [!code focus:10]
% asdf plugin add golang
% asdf install golang [版本]

# 2. 设置全局环境下的版本
% asdf global golang [版本]
% asdf reshim golang

# 3. 检查安装的版本
% go version
go version go1.22.5 darwin/arm64
```

:::
