# Gin

![](/static/skill-images/web-backend--gin.png)

Gin 是一个基于 Golang 的 Web 框架

## 下载安装

```zsh
% cd [项目名]                               # [!code focus:10]

# 1. 初始化项目模块
% go mod init [项目名]

# 2. 在项目中下载安装 Gin
% go get -u github.com/gin-gonic/gin

# 3. 检查 Gin 的版本
% go list -m github.com/gin-gonic/gin
github.com/gin-gonic/gin v1.10.0
```
